# 改命剧场：麦城前夜

这是一个适合公开发布到 GitHub Pages 的静态试玩 Demo。

## 当前目标

用一个耳熟能详的历史节点验证三件事：

1. 玩家会不会因为“改写意难平”点进来
2. 玩家会不会为了不同选择反复重开
3. 玩家会不会愿意把自己的结局截图或转发出去

## 本地打开

直接在浏览器中打开：

- `index.html`

或者在该目录运行：

```bash
python3 -m http.server 8080
```

然后访问：

- `http://127.0.0.1:8080`

## 当前剧本

- `麦城前夜`

## 公开策略

这版 Demo 使用历史 / 公版题材做公开验证，避免把受版权保护的影视小说 IP 直接放进公开仓库。
