# Contributing

感谢你愿意为“改命剧场”提供帮助。

## 当前最需要的贡献

1. 试玩反馈
2. 结局平衡建议
3. 更强的关键节点建议
4. 历史 / 公版题材的可公开剧本提案
5. UI 和文案打磨建议

## 提交前请注意

1. 不要提交受版权保护影视、动画、小说的直接改编内容到公开仓库
2. 不要提交未授权图片、海报、剧照、配乐或角色立绘
3. 如果加入第三方素材，必须同步更新 `ASSET_LICENSES.md`
4. 保持 Demo 的第一目标：让用户尽快玩到、看懂、愿意分享

## 推荐提交流程

1. 先开 Issue
2. 说明你想改什么
3. 如果涉及剧情，请写清楚这个节点想验证什么情绪或决策冲突
4. 再提交 PR

## 我们最欢迎的 Issue 类型

1. 这个选择不够痛，玩家没有犹豫感
2. 这个结局差异不够大，不值得二周目
3. 这个节点不够耳熟能详，传播性不够强
4. 这里的 UI 影响读剧情节奏
