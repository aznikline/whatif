## Primary Sources

1. Stanford course site: [CS146S: The Modern Software Developer](https://themodernsoftware.dev/)
   - Used for course description, prerequisites, weekly schedule, grading, and FAQ.
   - The site is client-rendered; the public bundle contains the schedule and reading lists.

2. Public assignments repository: [mihail911/modern-software-dev-assignments](https://github.com/mihail911/modern-software-dev-assignments)
   - Used for assignment shapes, tooling expectations, and practical deliverables.

## Verified Public Facts Used in Notes

- Units: 3
- Prerequisites: CS111 equivalent programming experience; CS221/229 recommended
- Format: weekly lectures, hands-on coding sessions, guest speakers, final project
- Grading:
  - Final Project: 80%
  - Weekly Assignments: 15%
  - Class Participation: 5%
- Estimated time commitment: 10-12 hours/week
- Language guidance: language-agnostic, but examples primarily use Python and JavaScript
